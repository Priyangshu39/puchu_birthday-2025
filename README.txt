
Interactive Birthday Site v2 - Instructions
------------------------------------------

This improved bundle fixes the cake animation freeze and upgrades the collage layout.
Drop audio files (optional) into the folder to enable sounds:
- blow.mp3
- cake_cut.mp3
- song.mp3

Files included:
- index.html
- styles.css
- script.js
- 7 image files (photos)
- README.txt

How to preview:
- Unzip and open index.html in a browser.
- If audio doesn't autoplay on mobile/desktop, use the Cut Cake button to start (it's a user gesture).

To publish: upload all files to your repo and enable GitHub Pages (branch: main, folder: / root).
